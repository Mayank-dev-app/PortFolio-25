### About My Portfolio

## Welcome to my Frontend Developer Portfolio!
This project showcases my skills in modern web development, focusing on clean design, performance, and responsive layouts.

I built this portfolio using React.js, applying best practices in UI/UX, component-based architecture, and reusability. The goal is to present my work, projects, and journey as a developer in an interactive and visually engaging way

## Features

Responsive and modern design

Interactive UI built with React + Tailwind CSS

Dynamic project showcase section

Smooth navigation and animations

Downloadable resume and contact form
